# AI Agent for Social Media Content Creation

This project contains an AI agent that generates social media content from YouTube videos.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/infoelite461-debug/my-agent.git
   cd my-agent
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the main script:
   ```bash
   python main.py
   ```
